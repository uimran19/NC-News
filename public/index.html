NC News API

  • GET /api
  • GET /api/topics
  • GET /api/articles
  • GET /api/articles/:article_id

GET /api

Serves up all the available endpoints of the api

GET /api/topics

Returns an array of topic objects

Each topic object contains:

  • slug → (string): The identifier for the topic
  • description → (string): A short description of the topic
  • img_url → (string): A link to an image representing the topic

Example Response:

                    {
                        "topics": [
                            {
                                "slug": "football",
                                "description": "Footie!",
                                "img_url": "https://images.pexels.com/photos/209841/pexels-photo-209841.jpeg?w=700&h=700"
                            }
                        ]
                    }

GET /api/articles

Returns an array of article objects

Each article object contains:

  • article_id → (number): The identifier for the article
  • title → (string): The title of the article
  • topic → (string): The topic name of the article
  • author → (string): The author of the article
  • created_at → (string): The article publication date
  • votes → (number): The number of votes given to an article
  • article_img_url → (string): A link to an image representing the article
  • comment_count → (number): The total number of comments on an article

Example Response:

                    {
                        "articles": [
                            {
                                "article_id": 33,
                                "title": "Seafood substitutions are increasing",
                                "topic": "cooking",
                                "author": "weegembump",
                                "created_at" "2018-05-30T15:59:13.341Z",
                                "votes": 0,
                                "article_img_url": "https://images.pexels.com/photos/1267320/pexels-photo-1267320.jpeg?w=700&h=700",
                                "comment_count": 6
                            }
                        ]
                    }

GET /api/users

Returns an array of user objects

Each article object contains:

  • username → (string): The username pertaining to a user
  • name → (string): The name of the user
  • avatar_url → (string): A link to

Example Response:

                    {
                        "users": [
                            {
                                "username": "tickle122",
                                "name": "Tom Tickle",
                                "avatar_url": "https://vignette.wikia.nocookie.net/mrmen/images/d/d6/Mr-Tickle-9a.png/revision/latest?cb=20180127221953"
                            }
                        ]
                    }
